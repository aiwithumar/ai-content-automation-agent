# AI Content Automation Agent

This project is an AI-powered automation workflow built using n8n.

## What This Project Does

- Automatically generates content ideas
- Creates captions and environments using AI
- Stores generated data in Google Sheets
- Converts ideas into cinematic video prompts

## Workflow Overview

1. Schedule Trigger starts the workflow
2. AI Agent generates content ideas
3. Data is stored in Google Sheets
4. Second AI Agent converts ideas into video prompts

## Tools Used

- n8n (Automation)
- OpenAI (AI Agent)
- Google Sheets (Data Storage)

## Features

- Fully automated workflow
- AI-generated content
- Structured data output
- Beginner-friendly automation project
